# Frontend Design Skill for Claude Code

## Description
UI/UX development tools and design patterns for frontend refinement, based on the anthropics/frontend-design skill. Enables Claude to generate, analyze, and improve frontend code and user interfaces.

## Installation
1. Visit: https://github.com/anthropics/skills/tree/main/skills/frontend-design
2. Copy the contents of the `frontend-design` skill folder into `.claude/skills/frontend-design`.
3. Ensure your Claude Code or MCP agent is configured to load skills from `.claude/skills/`.

## Usage
- Use this skill to generate or refine frontend code, apply design best practices, and improve UI/UX.
- Example prompt: `Refine the game's UI using the frontend-design skill for better accessibility and modern look.`
