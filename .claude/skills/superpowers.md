# Superpowers Skill for Claude Code

## Description
A comprehensive set of agent skills for development and testing, including systematic debugging, subagent-driven development, test-driven development, and more. Integrates workflows and methodologies from the obra/superpowers repository.

## Installation
1. Visit: https://github.com/obra/superpowers
2. Copy the relevant SKILL.md files (e.g., systematic-debugging, subagent-driven-development, test-driven-development) into `.claude/skills/`.
3. Ensure your Claude Code or MCP agent is configured to load skills from `.claude/skills/`.

## Usage
- Use these skills to automate and refine development, debugging, and testing workflows.
- Example prompt: `Use the systematic-debugging skill to analyze and resolve a bug in the game logic.`
